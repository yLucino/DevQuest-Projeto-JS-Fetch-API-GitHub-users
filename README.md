## Instalação

Instruções para instalar e configurar o projeto.

```sh
# Clone o repositório
git clone https://github.com/yLucino/DevQuest-Projeto-JS-Fetch-API-GitHub-users.git

# Navegue até o diretório do projeto
cd repo

# Instale as dependências
npm install
```
